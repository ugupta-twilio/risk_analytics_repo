# Skill Inventory

This folder contains the skill and capability inventory for the GM Risk Analytics area.

## Purpose

Tracks what analytical skills, models, and tools are available across the GM team's projects —
useful for onboarding new analysts and identifying reusable work.

## How to update

1. Add or update entries in this folder on a branch named `update/skill-inventory-<date>`
2. Open a PR — the GM ticket lead (`@ugupta-twilio`) is auto-assigned as reviewer
3. Merge once approved

## Access

This folder sits outside the per-ticket structure and is maintained by the ticket lead.
Analysts do not have write access here via the standard path-check rules.
