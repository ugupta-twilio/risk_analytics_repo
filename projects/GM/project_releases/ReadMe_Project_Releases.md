# GM Project Releases

This folder tracks released projects and skill inventory for the GM area of Risk Analytics.

## Contents

| Folder | Description |
|---|---|
| `skill_inventory/` | Inventory of skills and capabilities across GM projects |

## Access

This folder is managed by the ticket lead for the GM area (`@ugupta-twilio`).
Changes here require the lead's approval via PR.

## Adding a release

1. Create a subfolder under the relevant ticket (e.g. `projects/GM/RISK-XXXX/`)
2. Document the release in the ticket's README
3. Open a PR — the ticket lead is auto-assigned as reviewer
